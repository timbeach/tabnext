# tabnext Privacy Policy

**tabnext collects no data. None. Not a little. None.**

- No browsing history, URLs, or page titles — tabnext declares zero
  permissions and cannot read them even if it wanted to.
- No analytics, telemetry, or crash reporting.
- No network requests of any kind. The extension contains no code that
  talks to any server, and no remote code is loaded.
- No storage. Nothing is written to disk; the extension keeps a few
  window timestamps in memory while your browser is open and forgets
  them the moment it isn't.

What tabnext does, in full: it moves newly created tabs next to your
current tab, and it displays the number of open tabs on its toolbar
badge. Both happen entirely inside your browser.

The complete source code is about a hundred lines and can be audited in
a few minutes: https://github.com/timbeach/tabnext

If this policy ever changes (it won't), the change will appear in this
file's git history first.
